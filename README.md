# CLog

Logging tool written in C

## Building

To build the program into `./bin/`

```sh
$ make
```

To build and install the program into `/usr/bin/`

```sh
$ make
$ sudo make install
```

To clean build directory

```sh
$ make clean
```

To uninstall from `/usr/bin/`

```sh
$ sudo make uninstall
```
