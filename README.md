# CLog

Logging tool written in C

## Building

To build the program into `./bin/`

```sh
make
```

To build and install the program into `[wherever you're supposed to]`

```sh
make install
```

To clean build directory

```sh
make clean
```

To uninstall from `[wherever you're supposed to]`

```sh
make uninstall
```
